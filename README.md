# vuex-training

## Project setup
```
npm install
```
## setup tailwind css
```
npm install -D tailwindcss postcss autoprefixer
```
## setup json server
```
npm install json-server
```
## run json server until the project work
```
json-server --watch db.json
```
### Compiles and hot-reloads for development
```
npm run serve
```
